## El Reto de la Visualización en Tiempo Real

Mostrar datos de 10.000 sensores en tiempo real requiere arquitectura pensada: ni polling REST ni WebSocket
ingenuo escalan. La solución combinó WebSockets agrupados por zona geográfica y un frontend que virtualiza
las filas del grid.

## Stack y Decisiones

- **Astro para la capa estática:** landing, docs y login son páginas SSG con SEO perfecto.
- **SPA React para el dashboard:** una vez autenticado, el usuario entra en una isla React que gestiona
  el estado complejo del dashboard (filtros, zoom temporal, múltiples series).
- **WebSocket con backpressure:** el backend NestJS envía solo cambios diferenciales y aplica throttling
  por cliente. TimescaleDB permite consultas históricas eficientes con continuous aggregates.

## Resultados

Latencia de actualización `<500ms` para 10.000 sensores simultáneos, con un consumo de banda ancha por
cliente de solo 80KB/s. La virtualización del grid permite mostrar 100.000 filas sin degradar el FPS.
